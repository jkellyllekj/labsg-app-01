IMPLEMENTATION

1) Track current background index

2) Apply background via a helper
Use the SAME element currently used for the background (document.body or an existing background container).

Add:

const BG_TARGET = document.getElementById("bg") || document.body;

function applyBackgroundByIndex(index) {
  const url = backgroundImages[index];
  BG_TARGET.style.backgroundImage = `url(${url})`;
}

3) Manual cycle handler (user click only)
Add:

function cycleBackgroundManually() {
  const btn = document.getElementById("bgCycleBtn");
  if (!btn) return;

  btn.disabled = true;

  BG_TARGET.style.transition = "opacity 220ms ease";
  BG_TARGET.style.opacity = "0";

  setTimeout(() => {
    bgIndex = (bgIndex + 1) % backgroundImages.length;
    applyBackgroundByIndex(bgIndex);

    BG_TARGET.style.opacity = "1";

    setTimeout(() => {
      btn.disabled = false;
    }, 260);
  }, 220);
}

function wireBackgroundCycleButton() {
  const btn = document.getElementById("bgCycleBtn");
  if (!btn) return;
  btn.addEventListener("click", cycleBackgroundManually);
}

Call wireBackgroundCycleButton() once after the UI renders.
Do NOT call cycleBackgroundManually() anywhere else.
Do NOT add background logic to Generate.
